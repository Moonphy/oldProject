移动端提示浮窗

提示窗，可以直接调用无依赖，引入gfdialog.js文件即可。

调用方式有三种：

1.通过new Dialog实例显示

Dialog dome

new GFDialog({ title: 'dialog提示', autoShow: true, content: '要显示的内容' })


2.通过new Dialog实例显示指定结构窗口

指定结构如何第三种方法html一样，在gf-dialog样式后加上“no-auto”表示不会自动加载，要手动实例化。

new GFDialog({ view: document.getElementById('myd'), title: 'dialog提示', isTitle: false, autoShow: true, content: '要显示的内容' })


3.在页面html写以下结构，页面加载完就会显示

参数说明

content: 必填，显示内容

view: dialog dom，不填实例会自动创建、填写则表示指定实例内容

isTitle: 是否有标题，默认为true

autoShow: 实例化后是否自动显示，如果为false,测通过show方法显示窗体,默认值为false

hasCancel: 是否有取消按扭、默认是false

isMask: 是否有遮罩层，默认为true

title: 标题，isTitle为true时必填

save: 确认按扭事件function，选填
