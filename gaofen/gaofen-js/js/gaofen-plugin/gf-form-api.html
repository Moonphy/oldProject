通用跨域数据提交

引入gf-form-api.min.js文件即可。

使用方式：

直接引入gf-form-api.min.js，引入参数属性


例子如下(由于线上环境例子不能提交，但表单结构可用)：

表单参数

jsonp: 选填，如地址带有jsonp=false时采用跨域共享数据调用方式通过post方式提交表单，默认不传采用jsonp提交数据

gf: 选填，true时默认为高分表单数据结构提交，默认自动

auto: 选填，是否自动运行，默认自动

如果要主动调用，请在引入的script后加上auto=false的参数，然后自行调用GF_FORM_INIT()方法

表单dom id设为：gf_form

提交按扭submitId为：gf_form_sub

表单内要包括postUrl提交地址

如果要自定义表单dom ID 、提交ID之类可以auto=false，然后自行调用GF_FORM_INIT({id:'',submitId:''})

表单数据验证

在input、textarea、select的dom之上加上vrel属性，其值为ne、phone、number之类

ne：代表非空验证，可以在ne后加上min、max之类扩展非空属性，如果vrel="ne:min=3,max=8"意思是输入框不能为空的同时只能输入大于3个字及小于8个字的内容

phone：验证只能是手机

number：验证只能是数字

email：只能输入邮箱

同时写入多个验证类型如：vrel="ne|email"
