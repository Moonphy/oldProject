初始化微信JDK

首先引入weixin-init.min.js文件

调用方法如下：

1.一般页面调用(默认初始化分享功能)

1.一般页面调用(默认初始化分享功能)： 加载此文件，然后GWX.wxready(); 全局变量： var shareData = { //初始化配置 appId: '', timestamp: '', nonceStr: '', signature: '', //分享功能属性 title: '', // 分享标题 desc: '', // 分享描述 link: '', // 分享链接 imgUrl: ' // 分享图标 }

2.Gaofen框架页面调用

GWX.wxready(shareData);

参数说明

appId: appId串

timestamp: 必填，生成签名的时间戳

title: 分享标题

desc: 分享描述

link: 分享链接

imgUrl: 分享图标
